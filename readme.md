# Desktop timer

## Instructions
 * Timer app for supporting issue tracking system (JIRA, REDMINE etc)
 
 ## Tech stack
 
 * electron
 * Vue
 * HTML
 * jQuery

## Run/Build source


Prepaire for running app  
```
npm install
```
Running app

```
npm run start 
```
Build for windows
```
npm run build_windows
```
Build for mac
```
npm run build_mac
```
