# Desktop timer

## Instructions
 * Timer app for supporting issue tracking system (JIRA, REDMINE etc)
 
 ## Tech stack
 
 * electron
 * Vue
 * HTML
 * jQuery

## Run from source


Prepaire for running app  
```
npm install
```
Running app

```
npm run start 
```
